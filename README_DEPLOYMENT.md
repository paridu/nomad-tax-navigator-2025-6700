# Local Deployment Guide: Global Compliance Compass

This project uses an automated batch script to prepare your Windows environment for the Compass system.

## Prerequisites
1.  **Windows 10/11**
2.  **Docker Desktop:** Ensure it is running and WSL2 backend is enabled.
3.  **Python 3.9+:** Ensure "Add Python to PATH" was selected during installation.
4.  **Node.js 18+:** Latest LTS recommended.
